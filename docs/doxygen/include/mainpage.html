The OpenGL Objects Wrapper Library (glow) provides an object oriented C++ interface for OpenGL's programmable pipeline (3.0+). glow is a cross-platform library licenced under the MIT license. Download here

Dependencies

The following dev-libraries and programs need to be provided for correct CMake configuration:
  • C++11 compatible compiler (e.g. gcc >=4.7, MSVC >=2013)
  • CMake (>=2.8.9, better 2.8.12): http://www.cmake.org/
  • OpenGL C++ Bindings (glbinding >=0.1): https://github.com/hpicgs/glbinding
  • OpenGL Mathematics (GLM, 0.9.3 or 0.9.4): http://glm.g-truc.net/
  • Window and Context creation (GLFW >=3.0.3): http://www.glfw.org/ (optional, needed for glowwindow module and examples)
